Regression Linier#

Linear Regression (Regresi Linear) adalah suatu regresi linear yang digunakan untuk mengestimasi atau memprediksi hubungan antara dua variabel dalam penelitian kuantitatif. Dimana regresi linear ini mampu membuat satu asumsi tambahan yang mengkorelasikan antara variabel independen dan dependen melalui garis yang paling sesuai dari titik data garis lurus. Artinya, bukan kurva atau semacam faktor penelompokkan.

Regresi dilakukan pada data kontinu

Rumus Persamaan Linear

y = ax + b

keterangan :

y = variabel terikat

x = variabel bebas

a = koefisien/gradien

b = intercept/konstanta

import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression
from sklearn.datasets import make_regression
from matplotlib import pyplot
X_test, y_test = make_regression(n_samples=20, n_features=1, noise=10)
reg = LinearRegression().fit(X_test, y_test)
reg.score(X_test, y_test)
